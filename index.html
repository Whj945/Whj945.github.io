jQuery对表单的操作及更多应用

发表于 2019-11-05 | 分类于 java

一、下载安装包

 1.下载地址，使用jdk华为镜像地址：https://repo.huaweicloud.com/java/jdk，选择最新版本jdk-13

 2.默认安装

 3.生成jre目录，使用管理员身份打开jdk安装目录，执行命令
   bin\jlink.exe --module-path jmods --add-modules java.desktop --output jre

二、配置系统变量

 1.我的电脑

 JAVA_HOME:（Java的安装目录，默认：C:\Program Files\Java\jdk-13）

 CLASSPATH：.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tool.jar

 Path：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin

三、测试

 javac

 java -version
